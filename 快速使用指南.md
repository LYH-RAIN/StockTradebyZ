# 📈 量化选股系统 - 快速使用指南

## 🎯 核心理解

### ❌ 错误理解
```
"被选股系统选中 = 立即买入"
```

### ✅ 正确理解
```
"被选股系统选中 = 有潜力的候选股"
"查看K线图B信号 = 判断具体买点"
"最新B信号在1-2天内 = 较好的买入时机"
```

---

## 🚀 三步交易法

### 第1步：筛选候选股（选股系统）
```
访问 http://127.0.0.1:5000
点击【出坑战法】或【少妇战法】
查看选股结果 → 找到候选股票池
```

**示例结果**：
```
002990  ¥31.40  +0.87%
300350  ¥7.03   +1.15%
603528  ¥9.28   +1.20%
```

**作用**：从5000+股票缩小到几只

---

### 第2步：查看买卖信号（K线图）
```
点击感兴趣的股票
查看K线图中的B信号（绿色三角）
观察最新B信号的日期
```

**关键信息**：
- **B信号**：买入时机（绿色向上三角）
- **S信号**：卖出时机（红色向下三角）
- **信号日期**：非常重要！

---

### 第3步：判断买入时机（交易决策）
```
最新B信号在今天或昨天 → ✅ 立即关注，考虑买入
最新B信号在2-5天前   → ⚠️ 谨慎观察，等回调
最新B信号5天以上    → ❌ 暂不买入，等新信号
```

---

## 📋 检查清单

在决定买入之前，问自己：

- [ ] 这只股票被选股系统选中了吗？ → **筛选候选**
- [ ] K线图上有B信号吗？ → **确认买点**
- [ ] 最新B信号是什么时候？ → **判断时机**
- [ ] 距离最新B信号几天了？ → **评估风险**
- [ ] 从B信号至今涨了多少？ → **位置判断**

**全部满意 → 考虑买入**
**有疑虑 → 继续观察**

## 💡 实战案例

### 案例1：理想买点 ✅
```
11-21: 被选股系统选中 @ ¥6.50
11-21: K线图出现B信号 @ ¥6.50

→ 选股日期 = B信号日期
→ ✅ 立即买入，把握最佳时机！
```

### 案例2：谨慎追高 ⚠️
```
11-18: K线图出现B信号 @ ¥6.97
11-21: 被选股系统选中 @ ¥7.03 (+0.9%)

→ 距离B信号3天，已涨0.9%
→ ⚠️ 可能偏高，等回调或下一个B信号
```

### 案例3：等待买点 ❌
```
11-10: K线图出现B信号 @ ¥6.50
11-21: 被选股系统选中 @ ¥7.20 (+10.8%)

→ 距离B信号11天，已涨10.8%
→ ❌ 不建议买入，等待回调后新的B信号
```

## ⚙️ 两种战法对比

### 出坑战法（突破前高）
```
选股条件：接近前高 + 放量 + 站上MA10
B信号：价格在前高75%-105% + 放量1.2倍

适合：捕捉调整后的二次启动
风险：假突破
```

### 少妇战法（KDJ超卖）
```
选股条件：KDJ超卖转强 + 放量
B1信号：J<-20后回升 + 收阳线 + 放量

适合：低位反弹机会
风险：可能继续下跌
```

## 🎓 高级技巧

### 技巧1：结合多个信号
```
✅ B信号（今天） + 成交量放大 + 站上均线
   → 最强买点

⚠️ B信号（3天前） + 成交量萎缩
   → 动能减弱，谨慎

❌ 无B信号 + 高位滞涨
   → 不建议买入
```

### 技巧2：分批买入
```
第1批：B信号当天买入30%
第2批：次日确认不破位加仓30%
第3批：突破前高后加仓40%
```

### 技巧3：设置止损
```
止损位 = 买入价 × 95%（5%止损）

B信号买入 @ ¥10.00
止损位 @ ¥9.50
```

## ⚠️ 常见误区

### 误区1：盲目追涨
```
❌ "被选股系统选中就买"
✅ "看K线图确认有最新B信号再买"
```

### 误区2：忽略时机
```
❌ "看到B信号就买（不管几天前）"
✅ "只在B信号当天或次日买"
```

### 误区3：不设止损
```
❌ "买了就一直持有，等赚钱"
✅ "设置5%止损，及时止损"
```
