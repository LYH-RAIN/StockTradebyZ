## 🔍 B1信号判断标准

### 看到绿色▲后，再次确认4个条件：

#### ✅ 1. 趋势线位置（最重要！）
```
最佳：价格 > 红线 > 蓝线（多头排列）⭐⭐⭐⭐⭐
次佳：价格在两线之间震荡      ⭐⭐⭐
不做：价格 < 蓝线 < 红线（空头排列）❌
```

#### ✅ 2. KDJ指标（右下角图表）
```
✓ J值曾到达超卖区（粉色区域下方，<20）
✓ J值开始回升（从低位向上）
✓ K线金叉D线（蓝线上穿橙线）
```

#### ✅ 3. MACD指标（中下图表）
```
✓ DIF金叉DEA（蓝线上穿橙线）
✓ MACD柱由绿转红（由负转正）
✓ 零轴上方更好
```

#### ✅ 4. K线形态
```
✓ B1当天是阳线（红色K线）
✓ 实体较大（不是十字星）
✓ 没有长上影线
```

## 💰 B1买入操作流程

### Step 1: 发现B1信号
- K线图上有绿色▲标记
- 四大条件全部符合

### Step 2: 计算止损位
```
止损位 = B1当日K线最低价 - (2-5个价位)

例如：
B1当日最低价 = 10.50元
止损位 = 10.50 - 0.05 = 10.45元
```

### Step 3: 确定仓位
```
强趋势（红线和蓝线向上）：30-50%
中性趋势（横盘震荡）      ：10-20%
弱趋势（趋势线向下）      ：不做或观望
```

### Step 4: 下单入场
```
时机：B1当天收盘前或次日开盘
价格：市价单或限价单（不追高）
止损：立即设定止损单（10.45元）
```

### Step 5: 持仓管理
```
Day 1: 入场，设止损
Day 2: 观察是否出现B2（深绿▲）
     - 如出现B2：可加仓50%
     - 如跌破止损：立即离场
Day 3-7: 持续观察
     - 看到S1（红色▼）：至少减仓50%
     - 盈利>10%：可部分止盈
     - 跌破止损：无条件离场
```

## 🎯 B2加仓信号

### 什么是B2？
B1买入后，次日继续上涨，确认趋势转强。

### 识别方法
- K线图上有**深绿色▲ + "B2"**标记
- 出现在B1的下一个交易日
- 价格高于B1日收盘价

### 操作建议
```
✅ 适合加仓：
   - 已持有B1仓位
   - 趋势线向上
   - MACD持续金叉

❌ 不适合加仓：
   - 没有B1仓位（追高风险大）
   - 跳空高开>5%（容易回调）
   - 成交量萎缩
```

### 加仓比例
```
首次B1仓位：30%
B2加仓：15%（不超过首次50%）
总仓位：45%

止损位：仍保持在B1的最低价下方
```

## 🚨 S1卖出信号

### 什么是S1？
放量大阴线，表示获利盘涌出或主力出货。

### 识别方法
- K线图上有**红色▼ + "S1"**标记
- 当天收出大阴线（绿色，实体>3%）
- 成交量明显放大

### 操作建议
```
立即减仓50%或全部离场！

情况1：盈利中
→ 至少减仓50%，锁定利润

情况2：持平或小亏
→ 全部离场，观望为主

情况3：次日继续阴跌
→ 清仓离场，不留恋
```

## ⚠️ 纪律铁律

### 1. "只输一根K线"（核心！）
```
❌ 错误：跌破了，再等等看，可能会反弹
✅ 正确：跌破止损位，立即离场，不问原因

只输一根K线 = 最多亏3-5%
不止损 = 可能亏20-50%
```

### 2. 不做弱势中的B1
```
❌ 趋势线向下 + B1信号 = 假信号概率大
✅ 趋势线向上 + B1信号 = 成功率高
```

### 3. 不追高不抄底
```
❌ 连续涨停后追高
❌ 暴跌后抄底
✅ 等待B1信号出现再入场
```

### 4. 严格控制仓位
```
单只股票：≤ 50%（包括B1+B2）
单一战法：≤ 70%
现金留底：≥ 30%
```

## 🔧 常见问题

**Q1: 为什么有B1标记但趋势线向下？**  
A: B1算法只看KDJ和价格形态，不看趋势。必须你自己判断趋势！

**Q2: 没有B1标记可以买吗？**  
A: 可以，但风险更大。B1是系统给的参考，最终要你自己判断。

**Q3: B1后立即跌破止损怎么办？**  
A: 立即离场！这说明信号失效，只输一根K线（3-5%），不要幻想反弹。

**Q4: 可以提前买入等B1吗？**  
A: 不建议。B1是转强信号，提前买可能在超卖区被套更久。

**Q5: S1出现后次日又涨了，后悔了怎么办？**  
A: 不后悔！风险控制第一。宁可少赚，不能多亏。
